Este proyecto corresponde a una app desarrollada con **Next.js** y **TypeScript**.
La aplicación consume la API de Fake Store y muestra una lista de productos con una funcionalidad de búsqueda en tiempo real.

## Funcionalidades

- Consumo de productos desde la Fake Store API
- Visualización de imagen, título y precio de cada producto
- Búsqueda en tiempo real (no sensible a mayúsculas/minúsculas)
- Arquitectura basada en componentes
- Uso de React, Next.js y TypeScript

## Instalación

Asegúrate de tener instalado **Node.js (versión 18 o superior)**.

Clona el repositorio:
git clone

Installa las dependencias:
npm install

Ejecuta:
npm run dev 

y abre en el navegador: 
http://localhost:3000
